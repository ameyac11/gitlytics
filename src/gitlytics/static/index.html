Gitlytics Dashboard — Persist GitHub Metrics Beyond 14 Days

An interactive analytics platform to visualize and persist your historical GitHub repository traffic metrics, views, clones, and visitor referrers over time.

Access Modes

Gitlytics can be accessed using four distinct modes based on your requirements:

  • GitHub OAuth: Complete cloud dashboard with automated backups and workspace management.
  • Personal Access Token (PAT): Direct API integration running securely on your local environment.
  • CSV Upload: Back-port and upload raw CSV traffic data to preserve history offline.
  • Public Username: Explore any public GitHub profile metrics without requiring an account or token.

Key Features

  • Track and compare historical traffic trends across multiple repositories.
  • Analyze deduplicated visitor referrer domains and cloning statistics.
  • Generate custom SVG markdown badges, star growth graphs, and widgets for your profile.
  • Automate synchronization workflows with the official GitHub Action.
